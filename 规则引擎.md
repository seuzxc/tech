
Kie
==
```
KIE(knowledge is everything) 是一个聚集项目，只在将我们相关的技术放在一起，他也是我们项目之间共享的核心
它包含以下不同单相关的项目，为业务自动化和管理提供完成的解决方案组合：
1、Drools 是一个业务规则管理系统，具有向前链接和向后链接推理的规则引擎，允许快速可靠的评估业务规则和复杂的事件处理。一个规则引起也是创建专家系统的基本构建块，在人工智能中，该专家系统是模拟人类专家的决策能力的计算机系统
2、jBPM 是一个灵活的业务流程管理套件，允许你为你的业务目标建模，而建模只需描述达到这些目标所要执行的步骤。
3、OptaPlanner 是一种约束求解器，可以用户员工排班，车辆路径，任务分配和云优化等用例
4、Business center是一个功能奇缺的web应用程序，用户自定义业务规则和流程的可视化组合
5、UberFire 是一个几月web的工作台框架（首eclipse 复客户端的启发）
```
